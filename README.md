- k

- Iterm 2

- forgit

- Nerd font file

## How to install, configure

### brew

```
$ brew install ripgrep
$ brew install mcfly
$ brew install zplug
$ brew install autojump
$ brew install k
```

### `.zshrc`

- mcfly 설치 후 `eval "$(mcfly init zsh)"` `.zshrc`에 추가

- 아래 명령어로 zsh plugin들 모두 설치

```
$ zplug install
```

### `.vimrc`

- 아래 vim 명령어로 플러그인 설치

```
PlugInstall
```
